2.session

session是另一种记录客户状态的机制，不同的是Cookie保存在客户端浏览器中，而session保存在服务器上。

安装session
```sh
npm i express-session --save
```
案例：
```sh
app.use(session({
  resave: true,
  saveUninitialized: false,
  secret: '3nqr9xzx2438fgsdam4324n',
  cookie: {
      maxAge: 1000 * 60 * 30
  }
}));
```
