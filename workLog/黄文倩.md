 **git的基本操作**
 
 1.连接远程仓库
  ```sh
  git remote add origin 仓库地址
  ```
  
 2.本地仓库文件提交到远程库
 ```sh
  git push origin master
 ```
 3.获得远程库内容
 ```shell
  git pull
 ```
  
 4. 分支 master 设置为跟踪来自 origin 的远程分支 master
  git branch --set-upstream-to=origin/master master
